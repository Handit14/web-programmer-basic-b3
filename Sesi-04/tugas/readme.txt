1. Judul Algoritma: Menghitung Luas dan Keliling Lingkaran

2. Input:
    Panjang jari-jari lingkaran (r)
    phi (π)

3. Langkah kalkulasi:
Untuk menghitung luas lingkaran, gunakan rumus: luas = π x r x r
di mana π (phi) memiliki nilai konstan 3.14
Untuk menghitung keliling lingkaran, gunakan rumus: keliling = 2 x π x r

4. logic:
    var r = 10;

    var phi = 22 / 7;

    // Menghitung luas lingkaran
    var luas = phi * r * r;

    // Menghitung keliling lingkaran
    var keliling = phi * 2 * r;
